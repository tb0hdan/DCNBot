# Claude Instructions

## On Startup
- Always read docs/PROJECT_NOTES.md first to understand the project context and current state.
- Always keep project notes up to date with any changes or observations.
- Always use pytest for tests and store them alongside the code they test.

## Project Commands
- Lint: `make lint`
- MyPy type check: `make mypy`
- Test: `make test`

## Important Files
- docs/PROJECT_NOTES.md - Contains project overview, goals, and current implementation status
- README.md - Setup instructions and usage guidelines.
